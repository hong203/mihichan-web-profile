# Tiệm Gà Rán Mihi Chan

A cute web profile for Mihi Chan's art commissions shop, built with React (front-end) and Node.js (back-end).

## Features

- Cute avatar and bio
- Smooth image gallery for samples
- Detailed commissions menu with pricing for Honkai Star Rail and Genshin Impact
- Links to terms and samples

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```

2. Run the development server:
   ```
   npm run dev
   ```

3. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Adding Images

Place your images in the `public/images/` folder:
- `avatar.jpg` for the profile picture
- `image1.jpg`, `image2.jpg`, etc. for the samples gallery

## Back-end

The back-end is a simple Node.js server. To run:
```
npm run server
```

## Technologies

- Front-end: React, Vite
- Back-end: Node.js with Express
